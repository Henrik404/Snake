# 貪吃蛇（Snake）遊戲

一個使用 Python 和 Pygame 開發的多模式貪吃蛇遊戲，具有現代化視覺效果、多種遊戲模式和豐富的遊戲機制。

![貪吃蛇遊戲](https://github.com/Henrik404/Snake/raw/main/assets/images/icon.png)

## 📋 遊戲特色

- **三種遊戲模式**：單人遊戲、雙人對戰和玩家對電腦AI
- **漸變色蛇身**：蛇身採用漸變色彩，視覺效果更佳
- **多種食物類型**：
  - 普通蘋果（+1分）
  - 金蘋果（+5分）
  - 毒藥（-3分，蛇身縮短）
- **視窗大小可調整**：遊戲介面可根據螢幕大小自適應調整
- **音效系統**：包含吃食物、遊戲結束和選單選擇等音效
- **流暢的遊戲體驗**：精心調校的動畫和順暢的操控

## 🎮 如何遊玩

### 安裝需求

1. Python 3.6 或更高版本
2. Pygame 庫

```bash
pip install pygame
```

### 啟動遊戲

1. 克隆或下載此儲存庫
2. 在終端機中導航至遊戲目錄
3. 執行主程式

```bash
python main.py
```

### 遊戲控制

- **選單操作**：使用滑鼠點擊按鈕選擇遊戲模式
- **單人/玩家1**：
  - ↑ 向上移動
  - ↓ 向下移動
  - ← 向左移動
  - → 向右移動
- **玩家2**：
  - W 向上移動
  - S 向下移動
  - A 向左移動
  - D 向右移動
- **遊戲中操作**：
  - P 暫停/繼續遊戲
  - 遊戲結束後，按任意鍵返回主選單

## 🎲 遊戲模式說明

### 單人遊戲

- 控制一條綠色的蛇
- 收集食物增加分數和長度
- 撞牆或撞自己身體會導致遊戲結束

### 雙人對戰

- 玩家1控制綠色蛇，玩家2控制藍色蛇
- 頭對頭碰撞時，分數較高的蛇獲勝
- 可通過讓對手撞到自己的身體來獲勝
- 當雙方同時死亡，比較分數和存活時間決定勝負

### 電腦對戰

- 玩家控制綠色蛇，電腦控制藍色蛇
- AI會自動尋找食物並嘗試避開障礙物
- 挑戰自己能否打敗智慧的AI對手

## 🧩 遊戲元素

### 蛇（Snake）

- 玩家蛇身帶有漸變色，頭部有眼睛
- 蛇每吃一個食物會增長一節身體
- 蛇死亡後會變成灰色

### 食物（Food）

- **普通蘋果**：紅色，增加1分和1節長度，出現機率90%
- **金蘋果**：金色，增加5分和5節長度，出現機率3%
- **毒藥**：紫色，減少3分和3節長度，出現機率7%
- 食物在場上存在時間有限，會自動消失並重新生成

### AI對手

- AI會智慧地避開牆壁和蛇的身體
- 優先尋找普通食物，避免毒藥
- 在沒有安全路徑時會尋找最佳生存策略

## 🔧 技術實現

### 檔案結構

- **main.py**：遊戲主程序，處理遊戲循環和狀態
- **game.py**：遊戲邏輯類，處理蛇的移動、碰撞檢測等
- **objects.py**：定義遊戲中的各種物件（蛇、食物、按鈕、AI蛇）
- **settings.py**：包含所有遊戲常數、顏色和路徑設定

### 遊戲系統

- 使用網格系統實現碰撞檢測
- 實現了複雜的頭對頭碰撞邏輯，更公平判定勝負
- 食物生成機制確保不會出現在蛇身上
- 畫面大小可調整，自動適應不同尺寸的螢幕

## ⚙️ 自定義設置

如需調整遊戲參數，可修改 `settings.py` 檔案：
- 調整網格大小和遊戲區域
- 修改蛇的移動速度
- 自定義食物類型和出現機率
- 變更顏色、音效和字體
